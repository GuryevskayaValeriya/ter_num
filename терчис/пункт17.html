Пункт 17. Полная и приведенная системы вычетов

§4. Теория сравнений — Теория чисел

📖 Теория

Определение. Любое число из класса эквивалентности $\equiv_m$ называется вычетом по модулю $m$. Совокупность вычетов по одному из каждого класса — полная система вычетов (содержит $m$ чисел).
Пример (m = 5).
Наименьшие неотрицательные вычеты: 0, 1, 2, 3, 4
Абсолютно наименьшие вычеты: −2, −1, 0, 1, 2
Лемма 1. 1) Любые $m$ попарно несравнимых по $\bmod m$ чисел образуют полную систему вычетов.
2) Если $(a, m) = 1$, то $ax + b$ при пробегании полной системы вычетов $x$ тоже пробегает полную систему.
Определение. Приведенная система вычетов по модулю $m$ — это вычеты из полной системы, взаимно простые с $m$. Содержит $\varphi(m)$ элементов.
Пример (m = 42). Приведенная система: $1, 5, 11, 13, 17, 19, 23, 25, 29, 31, 37, 41$ (12 чисел, $\varphi(42) = 12$).
Лемма 2. Если $(a, m) = 1$ и $x$ пробегает приведенную систему вычетов, то $ax$ тоже пробегает приведенную систему.
Теорема 1 (суммы корней единицы). Пусть $x$ пробегает полную систему вычетов по модулю $m$. Тогда: $$\sum_{x} e^{2\pi i a x/m} = m, \text{ если } m \mid a, \text{ и } 0 \text{ в противном случае.}$$
Теорема 2 (сумма первообразных корней). $\sum_{\xi} e^{2\pi i \xi/m} = \mu(m)$, где суммирование по приведенной системе вычетов, $\mu$ — функция Мебиуса.
