Пункт 27. Число $\pi \approx 3{,}14159\ldots$

§5. Трансцендентные числа — Теория чисел

📖 Теория

Определение. Число $\pi$ — отношение длины окружности к её диаметру. Архимед вычислил, что $3 + \frac{10}{71} < \pi < 3 + \frac{1}{7}$, т.е. $3{,}1409 < \pi < 3{,}1429$.
Теорема 1. Число $\pi$ иррационально.
Идея доказательства. Используется тождество Эрмита с функцией $\sin x$:
$\int_0^\pi f(x)\sin x\, dx = F(\pi) + F(0)$, где $F(x) = f(x) - f''(x) + f^{(4)}(x) - \ldots$
При $f(x) = \frac{b^n x^n (a - bx)^n}{n!}$ интеграл $< 1$ при больших $n$, а $F(\pi) + F(0)$ — целое число. Противоречие.
Теорема 2 (Линдеман, 1882). Число $\pi$ трансцендентно.
Идея доказательства. Используется тождество Эрмита: $e^x F(x) - F(0) = \int_0^x e^{x-t} f(t)\, dt$. Если $\pi$ — алгебраическое, то $\gamma = i\pi$ — тоже алгебраическое. Пусть $\alpha_1, \ldots, \alpha_m$ — сопряжённые с $\gamma$. Тогда $1 + e^{\gamma_1} \cdots e^{\gamma_m} = 0$. При подходящем $f(x)$ левая часть — ненулевое целое, правая $< 1$. Противоречие.
Из доказательства трансцендентности $\pi$ следует неразрешимость древнегреческой задачи о квадратуре.circle: невозможно построить циркулем и линейкой отрезок длины $\pi$.
