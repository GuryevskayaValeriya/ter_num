§4. Теория сравнений — Теория чисел

📖 Теория

Теорема 1. Сравнение $x^{2} \equiv a \pmod{p}$ ($p$ — простое, $p \nmid a$) либо не имеет корней, либо имеет ровно два корня. Если $a$ — квадратичный вычет по $\bmod p$, то $x_{1} = a^{(p+1)/4} \pmod{p}$ — один из корней, второй: $p - x_{1}$.
Определение (Символ Лежандра). Для нечётного простого $p$ и произвольного $a$:
$(a/p) = 1$, если $a$ — квадратичный вычет $\bmod p$;
$(a/p) = -1$, если $a$ — квадратичный невычет $\bmod p$;
$(a/p) = 0$, если $p \mid a$.
Критерий Эйлера. $$a^{(p-1)/2} \equiv \left(\frac{a}{p}\right) \pmod{p}$$
Лемма Гаусса. Пусть $S$ — число элементов $r \in \{1, 2, \ldots, (p-1)/2\}$ таких, that $(ar \bmod p) > (p-1)/2$. Тогда $(a/p) = (-1)^{S}$.
Пример (таблица квадратичных вычетов).
$\bmod 7$: вычеты — $1, 2, 4$ (т.к. $1^{2}=1$, $2^{2}=4$, $3^{2}=2$), невычеты — $3, 5, 6$.
$(1/7) = 1$, $(2/7) = 1$, $(3/7) = -1$, $(4/7) = 1$, $(5/7) = -1$, $(6/7) = -1$.
Свойства символа Лежандра:
1) $a \equiv b \pmod{p} \Rightarrow (a/p) = (b/p)$
2) $(a_{1}a_{2}\ldots a_{n}/p) = (a_{1}/p)(a_{2}/p)\ldots(a_{n}/p)$
3) $(a^{2}/p) = 1$ при $p \nmid a$
4) $(ab^{2}/p) = (a/p)$
Дополнительные законы:
$\left(\frac{-1}{p}\right) = (-1)^{(p-1)/2}$
$\left(\frac{2}{p}\right) = (-1)^{(p^{2}-1)/8}$ — $2$ является кв. вычетом при $p \equiv 1, 7 \pmod{8}$.
