Пункт 4. Алгоритм Евклида

§1. Основные понятия и теоремы — Теория чисел

📖 Теория

Алгоритм Евклида — способ нахождения НОД двух чисел, основанный на свойстве 3 пункта 2: если a = bq + c, то (a, b) = (b, c).

Алгоритм:
1. Ввести a, b (a ≥ b ≥ 0).
2. Если b = 0, то Ответ: a. Конец.
3. r := остаток от деления a на b; a := b; b := r.
4. Перейти на шаг 2.

В виде цепочки равенств:

a = b·q1 + r1, 0 ≤ r1 < b
b = r1·q2 + r2, 0 ≤ r2 < r1
r1 = r2·q3 + r3, 0 ≤ r3 < r2
…
rn−2 = rn−1·qn + rn, 0 ≤ rn < rn−1
rn−1 = rn·qn+1 → (a,b) = rn
Линейное представление НОД. Из цепочки равенств алгоритма, поднимаясь снизу вверх, можно выразить (a, b) = a·u + b·v для некоторых целых u, v.
Пример. Найдём (525, 231):
525 = 231·2 + 63
231 = 63·3 + 42
63 = 42·1 + 21
42 = 21·2 + 0 → (525, 231) = 21

Линейное представление:
21 = 63 − 42·1 = 63 − (231 − 63·3) = 525·4 − 231·9 → u = 4, v = −9.
