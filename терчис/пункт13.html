Пункт 13. Мультипликативные функции

§3. Важнейшие функции в теории чисел — Теория чисел

📖 Теория

Определение. Функция $\theta : \mathbb{N} \to \mathbb{R}$ называется мультипликативной, если:
1) Существует $a \in \mathbb{N}$ такой, что $\theta(a) \neq 0$.
2) Для любых взаимно простых $a_1, a_2 \in \mathbb{N}$ выполняется:
$$\theta(a_1 \cdot a_2) = \theta(a_1) \cdot \theta(a_2)$$
Пример 1. $\theta(a) = a^s$, где $s$ — любое число.
Проверка: $(a_1 a_2)^s = a_1^s \cdot a_2^s$ при $(a_1, a_2) = 1$. ✓
Свойство 1. $\theta(1) = 1$ для любой мультипликативной функции.
Свойство 2. Если $a = p_1^{\alpha_1} p_2^{\alpha_2} \cdots p_n^{\alpha_n}$ — каноническое разложение, то:
$$\theta(a) = \theta(p_1^{\alpha_1}) \cdot \theta(p_2^{\alpha_2}) \cdots \theta(p_n^{\alpha_n})$$
Свойство 3 (обратное). Если задать $\theta(1) = 1$ и произвольно определить $\theta(p^{\alpha})$ для всех простых $p$ и натуральных $\alpha$, то единственным образом доопределяется мультипликативная функция на всех натуральных числах.
Свойство 4. Произведение нескольких мультипликативных функций является мультипликативной функцией.
Обозначение. Символом $\displaystyle\sum_{d \mid n}$ обозначается сумма по всем делителям $d$ числа $n$.
Лемма 1. Пусть $a = p_1^{\alpha_1} \cdots p_n^{\alpha_n}$, $\theta$ — мультипликативная функция. Тогда:
$$\sum_{d \mid a} \theta(d) = \prod_{i=1}^{n} \Bigl(\theta(1) + \theta(p_i) + \theta(p_i^2) + \ldots + \theta(p_i^{\alpha_i})\Bigr)$$
Лемма 2. Если $\theta(a)$ — мультипликативная функция, то функция
$$\chi(a) = \sum_{d \mid a} \theta(d)$$
также мультипликативна.
Пример 2. Пусть $\theta(1) = 1$ и $\theta(p^{\alpha}) = 2$ для всех $p$, $\alpha$.
Тогда $\theta(p_1^{\alpha_1} \cdots p_n^{\alpha_n}) = 2^n$ — число различных простых делителей.
