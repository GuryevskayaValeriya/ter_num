Пункт 5. Линейные диофантовы уравнения с двумя неизвестными

§1. Основные понятия и теоремы — Теория чисел

📖 Теория

Определение. Диофантово уравнение — уравнение (обычно с целыми коэффициентами), которое требуется решить в целых числах.

Рассмотрим линейное диофантово уравнение с двумя неизвестными:

ax + by = c,   a, b, c ∈ ℤ,   a и b ≠ 0
Условие разрешимости. Уравнение ax + by = c имеет решение в целых числах ⇔ (a, b) ∣ c.
Если (a, b) = d и d ∣ c, то после деления на d получаем уравнение с (a₁, b₁) = 1.

Случай 1. Однородное уравнение: ax + by = 0, (a, b) = 1.

Общее решение: x = −b·t,   y = a·t,   t ∈ ℤ.

Случай 2. Неоднородное уравнение: ax + by = c, (a, b) = 1.

Пусть (x₀, y₀) — частное решение. Тогда общее решение:
x = x₀ − b·t
y = y₀ + a·t
где t ∈ ℤ — произвольный параметр.
Как найти частное решение. Так как (a, b) = 1, то ∃ u, v ∈ ℤ: au + bv = 1 (алгоритм Евклида).
Умножаем на c: a(uc) + b(vc) = c → x₀ = uc, y₀ = vc.
Пример. Решим 7x + 12y = 43.
Алгоритм Евклида: 12 = 7·1 + 5, 7 = 5·1 + 2, 5 = 2·2 + 1 → (7,12) = 1.
1 = 5 − 2·2 = (12−7) − (7−(12−7))·2 = 12·3 + 7·(−5) → u = −5, v = 3.
x₀ = −5·43 = −215, y₀ = 3·43 = 129.
Общее решение: x = −215 − 12t, y = 129 + 7t.
При t = −18 получаем x = 1, y = 3.
