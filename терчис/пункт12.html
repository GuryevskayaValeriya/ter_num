Пункт 12. Целая и дробная часть

§3. Важнейшие функции в теории чисел — Теория чисел

📖 Теория

Определение. Пусть $x \in \mathbb{R}$ — действительное число.
• Целой частью $\lfloor x \rfloor$ числа $x$ называется его нижнее целое, т.е. наибольшее целое, не превосходящее $x$.
• Дробной частью $\{x\}$ числа $x$ называется число $\{x\} = x - \lfloor x \rfloor$.
Примеры.
$\lfloor 2{,}81 \rfloor = 2$ ; $\{2{,}81\} = 0{,}81$
$\lfloor -0{,}2 \rfloor = -1$ ; $\{-0{,}2\} = 0{,}8$
Свойства:
• Целая часть — неубывающая функция.
• Дробная часть — периодическая с периодом 1, всегда $0 \leq \{x\} < 1$.
• Обе функции разрывны при целых x, но непрерывны справа.
Лемма 1 (формула Лежандра). Показатель, с которым простое число p входит в разложение n!, равен:
$$\alpha = \left\lfloor \frac{n}{p} \right\rfloor + \left\lfloor \frac{n}{p^2} \right\rfloor + \left\lfloor \frac{n}{p^3} \right\rfloor + \ldots$$
Пример. Показатель 5 в 643!:
$\lfloor 643/5 \rfloor + \lfloor 643/25 \rfloor + \lfloor 643/125 \rfloor + \lfloor 643/625 \rfloor = 128 + 25 + 5 + 1 = $ 159.
Определение. Точка координатной плоскости называется целой, если обе её координаты — целые числа.
Лемма 2. Пусть функция f(x) непрерывна и неотрицательна на отрезке [a, b]. Тогда число целых точек в области D = {a < x ≤ b, 0 < y ≤ f(x)} равно:
$$\sum_{\substack{x \in \mathbb{Z} \\ a < x \leq b}} \lfloor f(x) \rfloor$$
Лемма 3 (формула площади многоугольника). Пусть $M$ — многоугольник на координатной плоскости с вершинами в целых точках, контур не пересекает и не касает сам себя, $S$ — площадь, $T = \sum_{A}(1 - \delta_A)$, где суммирование по всем целым точкам $A$ внутри и на границе, $\delta_A = 1$ для внутренних, $\delta_A = 1/2$ для граничных. Тогда $T = S$.
Теорема (Дирихле). Для любого $\alpha \in \mathbb{R}$ число $0$ является предельной точкой последовательности $x_n = \{\alpha \cdot n\}$.
Следствие. Если $\alpha \in \mathbb{R}$ иррационально, то члены последовательности $x_n = \{\alpha \cdot n\}$ всюду плотно заполняют отрезок $[0, 1]$.
