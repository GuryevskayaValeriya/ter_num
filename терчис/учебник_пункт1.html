§1. Основные понятия и теоремы — Теория чисел

📖 Теория

Определение. Пусть a, b ∈ ℤ. Число a делится на число b, если найдётся такое число q ∈ ℤ, что a = q·b.
Синонимы: a кратно b; b — делитель a.
Запись: a ⋮ b или b | a.

Отношение делимости на множестве натуральных чисел ℕ рефлексивно, антисимметрично и транзитивно — то есть является отношением частичного порядка.

Теорема (о делении с остатком). Для данного целого отличного от нуля числа b всякое целое число a единственным образом представимо в виде
a = b·q + r,   где   0 ≤ r < |b|.
Определение. Число q называется неполным частным, а число r — остатком от деления a на b.
Важно! Остаток всегда неотрицательный (0 ≤ r < |b|). Неполное частное может быть любым целым числом.
Пример 1. Разделим 23 на 5:
23 = 5·4 + 3 → неполное частное 4, остаток 3.

Пример 2. Разделим −23 на 5:
Надо подобрать q так, чтобы остаток был 0 ≤ r < 5.
−23 = 5·(−5) + 2 → неполное частное −5, остаток 2.
(Проверка: 5·(−5) + 2 = −25 + 2 = −23.)
Не путайте: −23 = 5·(−4) − 3 — это неверно, так как остаток −3 < 0!
Пример 3. Разделим 17 на −6:
17 = (−6)·(−2) + 5 → неполное частное −2, остаток 5.
Проверка: (−6)·(−2) + 5 = 12 + 5 = 17. ✓
